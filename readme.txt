Activate Google Analytics on your website.

== Description ==

Simple but efficient plugin to activate Google Analytics on your website.
Some options allows you to choose which content should be tracked.
http://www.google.com/analytics

== Installation ==

1. Extract the plugin to your plugins directory.
2. Activate the plugin in the plugins page.
3. Configure the plugin on the admin settings page.
4. Enjoy your stats on Google Analytics
